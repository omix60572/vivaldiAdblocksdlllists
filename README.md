### Статические фильтры для использования в блокировщиках совместимыми с uBlock Origin / Brave Shields / Vivaldi Browser internal AdBlocker

### Списки для подключения:
1) customFilters.txt - обычный набор фильтров, который с меньшей вероятностью сломает какой-либо сайт
2) customFiltersStrict.txt - набор фильтров, который может сломать сайт или убрать те элементы которые могут быть кому-нибудь полезны
3) customFiltersMobile.txt - набор фильтров, для использования в мобильных браузерах
